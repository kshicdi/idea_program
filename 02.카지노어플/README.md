# 카지노 룰렛 게임 - 마틴게일 배팅 계산기

웹 기반 룰렛 게임 배팅 계산 도구입니다. 마틴게일법을 사용하여 배팅 금액과 위치를 자동으로 계산합니다.

## 기능

- **세 가지 게임 타입**
  - 대소 게임 (19-36 vs 1-18)
  - 블랙/레드 게임
  - 홀짝 게임

- **줄서기 (대기열) 기능**
  - 각 게임별로 독립적인 대기열 관리
  - 배팅 위치를 순서대로 관리

- **마틴게일 배팅 시스템**
  - 패배 시 배팅 금액 2배 증가
  - 승리 시 초기 배팅 금액으로 리셋
  - 다음 배팅 금액과 위치 자동 계산

- **배팅 히스토리**
  - 모든 배팅 기록 저장 및 조회
  - 승리/패배 결과 표시

## 사용 방법

### 방법 1: Render에 배포하기 (권장) 🌐

**PC를 켜두지 않아도 어디서든 접속 가능합니다!**

1. **Render 배포 가이드 참고**
   - `RENDER_DEPLOY.md` 파일을 참고하여 Render에 배포
   - GitHub에 코드 업로드 후 Render에서 웹 서비스 생성
   - 배포 완료 후 제공되는 URL로 어디서든 접속 가능

2. **장점**
   - PC를 켜두지 않아도 24시간 접속 가능
   - 모바일에서도 어디서든 접속 가능
   - 무료 티어 사용 가능 (15분 비활성 시 sleep)

자세한 배포 방법은 `RENDER_DEPLOY.md` 파일을 참고하세요.

### 방법 2: 로컬에서 실행하기

1. 터미널에서 다음 명령어 실행:
```bash
cd "02.카지노어플"
python3 app.py
```

2. 브라우저가 자동으로 열리거나, 수동으로 `http://localhost:3822` 접속

### 방법 3: 인터넷에서 접속하기 (ngrok 사용) 🌐

1. **ngrok 설치** (처음 한 번만):
   - https://ngrok.com/download 에서 다운로드
   - 또는 Homebrew 사용: `brew install ngrok`
   - ngrok 계정 생성 및 인증 토큰 설정 (무료 계정 사용 가능)

2. **서버 실행**: 
   ```bash
   cd "02.카지노어플"
   python3 app.py
   ```

3. **인터넷 접속 주소 확인**:
   - 서버 실행 시 터미널에 표시되는 "인터넷 접속" 주소를 확인합니다.
   - 예: `https://xxxx-xx-xx-xx-xx.ngrok-free.app`
   - 이 주소는 어디서든 접속 가능합니다 (같은 WiFi가 아니어도 됩니다!)

4. **모바일 또는 다른 기기에서 접속**:
   - 웹 브라우저에서 위의 인터넷 접속 주소를 입력합니다.
   - 어디서든 접속 가능합니다!

**참고**: 
- ngrok 무료 버전은 세션당 2시간 제한이 있습니다.
- ngrok이 설치되어 있지 않으면 로컬/네트워크 접속만 가능합니다.
- **PC가 켜져 있어야 접속 가능합니다.**

### 방법 4: 같은 WiFi 네트워크에서 접속하기

1. **서버 실행**: 방법 1로 서버를 실행합니다.

2. **같은 WiFi 네트워크 확인**: 
   - 컴퓨터와 모바일 기기가 같은 WiFi 네트워크에 연결되어 있어야 합니다.

3. **IP 주소 확인**: 
   - 서버 실행 시 터미널에 표시되는 "네트워크 접속" 주소를 확인합니다.
   - 예: `http://192.168.0.100:3822`

4. **모바일 브라우저에서 접속**:
   - 모바일 기기의 웹 브라우저(Chrome, Safari 등)를 엽니다.
   - 주소창에 표시된 네트워크 접속 주소를 입력합니다.

**참고**: 
- 방화벽이 활성화되어 있으면 접속이 차단될 수 있습니다. 필요시 방화벽 설정을 확인하세요.
- 컴퓨터의 IP 주소가 변경되면 새로운 주소로 접속해야 합니다.
- **PC가 켜져 있어야 접속 가능합니다.**

### 방법 5: 직접 HTML 파일 열기

`index.html` 파일을 더블클릭하여 브라우저에서 직접 열 수 있습니다. (모바일에서는 이 방법을 사용할 수 없습니다)

## 게임 사용법

1. **초기 배팅 금액 설정**
   - 각 게임의 "초기 배팅 금액" 입력란에 원하는 금액 입력

2. **대기열 추가**
   - "대기열 추가" 버튼을 클릭하여 배팅 위치를 순서대로 추가
   - 첫 번째 클릭: 첫 번째 옵션 추가 (예: 대, 블랙, 홀)
   - 두 번째 클릭: 반대 옵션 추가 (예: 소, 레드, 짝)
   - 이후 클릭: 이전 항목과 반대 옵션이 자동으로 추가됨

3. **배팅 결과 입력**
   - "승리" 버튼: 배팅이 성공했을 때
     - 초기 배팅 금액으로 리셋
     - 현재 대기열 항목 제거
   - "패배" 버튼: 배팅이 실패했을 때
     - 배팅 금액 2배 증가 (마틴게일법)
     - 다음 대기열 항목으로 이동

4. **배팅 히스토리 확인**
   - 하단의 히스토리 섹션에서 모든 배팅 기록 확인
   - 탭을 클릭하여 각 게임별 기록 조회

## 마틴게일법 설명

마틴게일법은 다음과 같이 작동합니다:

- **첫 배팅**: 초기 배팅 금액 (예: 100)
- **패배 시**: 이전 배팅 금액의 2배 (100 → 200 → 400 → 800 ...)
- **승리 시**: 초기 배팅 금액으로 리셋 (100)

예시:
- 1차: 100원 배팅 → 패배
- 2차: 200원 배팅 → 패배
- 3차: 400원 배팅 → 승리
- 4차: 100원 배팅 (리셋)
