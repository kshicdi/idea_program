# 안드로이드 파일 전송 프로그램

안드로이드 기기에 USB 또는 WiFi를 통해 파일을 쉽게 전송할 수 있는 프로그램입니다.

## 전송 방식

### 1. WiFi/FTP 방식 (추천 - USB 디버깅 불필요)
- USB 디버깅 설정이 필요 없습니다.
- 안드로이드와 컴퓨터가 같은 WiFi 네트워크에 연결되어 있으면 됩니다.
- 안드로이드에서 FTP 서버 앱만 실행하면 됩니다.

### 2. USB/ADB 방식
- USB 디버깅이 활성화되어 있어야 합니다.
- ADB가 설치되어 있어야 합니다.

## 필요 사항

1. **Python 3.6 이상**
   - Python이 설치되어 있어야 합니다.
   - tkinter, ftplib는 Python 기본 라이브러리이므로 별도 설치가 필요 없습니다.

2. **WiFi/FTP 방식 사용 시**
   - 안드로이드와 컴퓨터가 같은 WiFi 네트워크에 연결
   - 안드로이드에서 FTP 서버 앱 설치 및 실행
     - 추천 앱: "FTP Server", "WiFi FTP Server", "FTP Server (Free)" 등
     - Google Play Store에서 "FTP Server"로 검색하면 여러 앱을 찾을 수 있습니다.

3. **USB/ADB 방식 사용 시**
   - ADB (Android Debug Bridge) 설치
     - Android SDK Platform Tools를 설치해야 합니다.
     - 다운로드: https://developer.android.com/studio/releases/platform-tools
     - 설치 후 시스템 PATH에 추가해야 합니다.
   - 안드로이드 기기 설정
     - 개발자 옵션 활성화
     - USB 디버깅 활성화
     - USB로 컴퓨터에 연결

## 사용 방법

### WiFi/FTP 방식 (USB 디버깅 불필요)

1. **안드로이드에서 FTP 서버 앱 실행**
   - Google Play Store에서 "FTP Server" 앱을 다운로드하여 설치합니다.
   - 앱을 실행하고 서버를 시작합니다.
   - 앱 화면에 표시되는 IP 주소, 포트, 사용자명, 비밀번호를 확인합니다.

2. **프로그램 실행**
   ```bash
   python android_file_transfer.py
   ```

3. **전송 방식 선택**
   - "WiFi/FTP (USB 디버깅 불필요)" 옵션을 선택합니다.

4. **FTP 서버 정보 입력**
   - 안드로이드 FTP 서버 앱에서 확인한 정보를 입력합니다.
     - IP 주소: 예) 192.168.0.100
     - 포트: 예) 2121 (기본값)
     - 사용자명: 예) android (앱에 따라 다를 수 있음)
     - 비밀번호: 예) android (앱에 따라 다를 수 있음)

5. **연결 확인**
   - "연결 확인" 버튼을 클릭하여 FTP 서버에 연결할 수 있는지 확인합니다.
   - "FTP 서버 연결됨" 메시지가 표시되면 준비 완료입니다.

6. **파일 선택 및 전송**
   - "파일 선택" 버튼으로 전송할 파일을 선택합니다.
   - 전송 경로를 입력합니다 (예: /Download, /sdcard/Download 등)
   - "파일 전송" 버튼을 클릭합니다.

### USB/ADB 방식

1. **프로그램 실행**
   ```bash
   python android_file_transfer.py
   ```

2. **전송 방식 선택**
   - "USB/ADB (USB 디버깅 필요)" 옵션을 선택합니다.

3. **기기 연결 확인**
   - USB로 안드로이드 기기를 컴퓨터에 연결합니다.
   - "연결 확인" 버튼을 눌러 기기 연결 상태를 확인합니다.

4. **파일 선택 및 전송**
   - "파일 선택" 버튼으로 전송할 파일을 선택합니다.
   - 전송 경로를 입력합니다 (기본값: `/sdcard/Download`)
   - "파일 전송" 버튼을 클릭합니다.

## 주의 사항

### WiFi/FTP 방식
- 안드로이드와 컴퓨터가 같은 WiFi 네트워크에 연결되어 있어야 합니다.
- 안드로이드에서 FTP 서버 앱이 실행 중이어야 합니다.
- 방화벽이 FTP 포트를 차단하지 않는지 확인하세요.
- 전송 경로는 FTP 서버 앱에서 접근 가능한 경로여야 합니다.

### USB/ADB 방식
- USB 디버깅이 활성화되어 있어야 합니다.
- 기기가 컴퓨터에 USB로 연결되어 있어야 합니다.
- 처음 연결 시 기기에서 USB 디버깅 허용을 확인해야 할 수 있습니다.
- 전송 경로는 안드로이드 기기에서 접근 가능한 경로여야 합니다.

## 문제 해결

### WiFi/FTP 방식

#### FTP 서버에 연결할 수 없는 경우
- 안드로이드와 컴퓨터가 같은 WiFi 네트워크에 연결되어 있는지 확인하세요.
- 안드로이드에서 FTP 서버 앱이 실행 중인지 확인하세요.
- IP 주소, 포트, 사용자명, 비밀번호가 올바른지 확인하세요.
- 방화벽이 FTP 포트를 차단하지 않는지 확인하세요.
- 안드로이드 기기의 IP 주소가 변경되었을 수 있습니다. FTP 서버 앱에서 현재 IP 주소를 다시 확인하세요.

#### 파일 전송이 실패하는 경우
- 전송 경로가 올바른지 확인하세요 (예: /Download, /sdcard/Download)
- 안드로이드 기기의 저장 공간이 충분한지 확인하세요.
- 파일 이름에 특수문자가 포함되어 있지 않은지 확인하세요.

### USB/ADB 방식

#### ADB를 찾을 수 없는 경우
- Android SDK Platform Tools를 설치하고 PATH에 추가했는지 확인하세요.
- 터미널에서 `adb version` 명령어로 ADB가 정상적으로 작동하는지 확인하세요.

#### 기기가 연결되지 않는 경우
- USB 케이블이 정상적으로 연결되어 있는지 확인하세요.
- USB 디버깅이 활성화되어 있는지 확인하세요.
- 기기에서 USB 디버깅 허용 팝업이 나타나면 허용을 선택하세요.
- USB 연결 모드를 "파일 전송" 또는 "MTP" 모드로 설정하세요.
